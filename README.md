  ## Running the code
  
  1. Run `npm i` to install the dependencies.
  2. Copy `.env.example` to `.env` and add your Google and GitHub OAuth credentials:
     ```bash
     cp .env.example .env
     ```
  3. Add your OAuth credentials to the `.env` file:
     - Get Google OAuth Client ID from [Google Cloud Console](https://console.cloud.google.com/)
     - Get GitHub OAuth Client ID from [GitHub Developer Settings](https://github.com/settings/developers)
  4. Run `npm run dev` to start the development server.
